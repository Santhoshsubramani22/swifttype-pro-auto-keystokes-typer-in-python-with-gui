
# 🚀 SwiftType Pro – Auto Keystroke Typer (Python GUI)

SwiftType Pro is a powerful and user-friendly Auto Keystroke Typer built using Python with Tkinter GUI.  
It allows users to automatically type large amounts of text with customizable delay and typing speed.

Developed by **Santhosh.s**

---

## 📌 Features

✅ Simple and Clean GUI  
✅ Custom Start Delay (Countdown before typing starts)  
✅ Adjustable Typing Speed  
✅ Pause / Resume Functionality  
✅ Stop Typing Anytime  
✅ Restart Application Instantly  
✅ User-Friendly Error Handling  

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- PyAutoGUI (for auto typing)
- Threading (for smooth background execution)

## 🎮 How It Works

1. Enter the text you want to auto-type.
2. Set the delay (countdown before typing starts).
3. Set the typing speed (seconds per character).
4. Click **Start**.
5. Switch to any application (Notepad, Browser, etc.).
6. The program will start typing automatically.

---

## 🎛 Controls

| Button       | Function                  |
| ------------ | ------------------------- |
| Start        | Begins the typing process |
| Pause/Resume | Temporarily pauses typing |
| Stop         | Stops typing immediately  |
| Restart      | Clears all inputs         |
| Exit         | Closes the application    |

---

## ⚠️ Important Note

⚠️ After clicking **Start**, quickly switch to the window where you want the text to be typed before the countdown ends.
